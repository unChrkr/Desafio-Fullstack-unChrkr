## Descrição:

Esse trabalho trata-se de uma plataforma para realização de cadastro de usuário. Assim que esse usuário é cadastrado e feito o login, 
existe a possibilidade de criar, deletar, fazer update e ver todos os contatos disponíveis. Também existe a possibilidade de update, deleção de usuário.

## Instalação
1. Faça o clone do projeto em sua máquina:
2. git clone  https://github.com/unChrkr/Desafio-Fullstack-unChrkr.git

## Frontend
4. Navegue para a pasta do frontend:
cd frontend
cd 'fronten part'
3. Instale as dependências do frontend:
yarn install ou npm install

## Backend
4. Navegue para a pasta do backend:
cd backend
5. Instale as dependências do backend:
yarn install ou npm install

O frontend estará disponível em: [http://localhost:3000](http://localhost:3000).
O backend estará disponível em: [http://localhost:3001](http://localhost:3001).
### Frontend
6. Para executar o frontend, utilize o seguinte comando:
yarn dev ou npm run dev

O frontend estará disponível em: [http://localhost:3000](http://localhost:3000).
### Backend
7. Para executar o backend em modo de desenvolvimento, utilize o seguinte comando:
yarn start:dev ou npm run start:dev

O backend estará disponível em: [http://localhost:3001](http://localhost:3001).
